    1. 精确模式

  1. 文件和数据格式化

    1. 文件的使用

      文件的打开关闭
      • 打开–操作–关闭
      • 打开–open < name >=open(< name >,< mode >) mode指文本或二进制
      • <>.name() 表示文件的名字
      • 默认值为 ‘rt’
      • 七种打开模式如下
      • ‘r’ 只读 若不存在返回 FileNotFoundError
      • ‘w’ 覆盖写 不存在则创建
      • ‘x’ 创建写 已存在则返回 FileExistsError
      • ‘a’ 追加写 不存在则创建
      • ‘b’ 二进制文件模式 放到 r w x a 后面
      • ‘t’ 文本文件模式 同 b
      • ‘+’ 与r/w/x/a一同使用，同时读写
      • close() 文件关闭
      文件内容的读取

      大文件如果一次性全部读取，时间和空间占用太大，最好分行处理或其他方法

      文件内容的写入

      • .write(s) s是字符串

      • .writelines(list) 参数为元素全为字符串的列表，但是是将各个元素拼接起来写入！
        没有换行没有空格

      • .seek(offset) 参数为0，1，2

        0表示将光标移动到整个文件开头，1表示当前位置，2表示文件末尾

        offset前面还可以加一个参数为光标移动几个单位

        但是如果光标移动到某个地方，再写入会把原来的内容覆盖掉

      • 同时如果想要将文件写入后再进行读取或其他操作，要用.seek()移动光标至开头
